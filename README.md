# Persian-Arabic-handwritten-digits-recognition
The code shows is Python implentation of the paper.

### Dataset

We have used HODA dataset.

Dataset paper:

https://www.sciencedirect.com/science/article/abs/pii/S0167865507000037

http://farsiocr.ir/

### Dependencies

Python 3.6 or newer.

* numpy
* pickle
* scipy
* pandas
* OpenCV
* scikit-learn 
* scikit-image
* matplotlib
* Plotly
